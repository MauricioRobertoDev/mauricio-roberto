## Sobre o projeto

O Galaxy Calculadora é uma ferramenta desenvolvida para usuários do
[Galaxy Chat & Play](https://galaxy.mobstudio.ru/web/), oferecendo uma solução eficiente para
calcular a autoridade necessária e estratégias no jogo "canhões e barris". Desenvolvido utilizando
VueJS 3 e Vite 4.

Eu criei esse projeto a muito tempo atrás, bem no começo do meu hoob, então refiz ele pra reaprender
como funcionada o VueJS e agora usando Typescript e Tailwindcss,

## Tecnologias utilizadas

-   [VueJS 3](https://vuejs.org/): Um framework JavaScript progressivo para a construção de
    interfaces de usuário.
-   [Vite 4](https://vitejs.dev/): Um construtor de aplicativos web extremamente rápido e flexível.
-   [ESlint](https://eslint.org/) e [Prettier](https://prettier.io/): Ferramentas para manter o
    código limpo, consistente e de alta qualidade.
-   [Tailwind CSS](https://tailwindcss.com/): Um framework de utilitários-first CSS para criar
    designs modernos e responsivos.
-   [TypeScript](https://www.typescriptlang.org/): Superset do JavaScript que adiciona tipagem
    estática para facilitar o desenvolvimento e a manutenção do código.

## Funcionalidades em destaque

-   Responsividade: A aplicação se adapta a diferentes dispositivos, proporcionando uma experiência
    consistente.
-   Cálculo de autoridade: Descubra a autoridade necessária para atingir seus objetivos no jogo.
-   Acertos: Saiba quantos acertos com cada tipo de bala são necessários para alcançar seus
    objetivos.

## Como usar

1. Digite o número de pontos da sua autoridade atual.
2. Selecione a autoridade desejada.

## Desenvolvimento Local

1. Clone o projeto:
    ```shell
    git clone https://github.com/mauriciorobertodev/galaxy-calculadora.git
    ```
2. Entre na pasta do projeto:
    ```shell
    cd galaxy-calculadora
    ```
3. Instale as dependências:
    ```shell
    npm install
    ```
4. Execute o modo de desenvolvimento:
    ```shell
    npm run dev
    ```
5. Acesse a url de desenvolvimento: [http://localhost:5173](http://localhost:5173).
